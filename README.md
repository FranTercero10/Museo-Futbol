El proyecto Museo del Fútbol consiste en una aplicación web estática que simula un museo virtual dedicado al fútbol. A través de distintas escenas interactivas, el usuario puede recorrer el museo y visualizar contenidos visuales relacionados con este deporte.

El objetivo principal del proyecto es aplicar conocimientos de HTML, CSS y Visual Studio Code para crear una experiencia visual e interactiva accesible desde un navegador web.
El proyecto se organiza de la siguiente forma:

index.html
Página principal del museo. Sirve como punto de entrada a la experiencia.

escena-360.html
Escena interactiva en 360º que permite una vista envolvente del museo.

escena-2-y-3.html
Página que contiene otras escenas o salas adicionales del museo.

assets/images/
Carpeta que almacena todas las imágenes utilizadas en el proyecto.

js/
Carpeta que contiene los archivos JavaScript encargados de la funcionalidad interactiva.
